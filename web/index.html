Technical Documentation Assistant

Cited, reliable answers over your ETL/SQL/Snowflake docs with RAG, summarization, OCR, and optional Pinecone.

Quickstart (local)

Run FastAPI (backend) and Streamlit (UI) in two terminals.

# 1) Create & activate venv
python -m venv .venv
source .venv/bin/activate

# 2) Install deps
pip install -r requirements.txt

# 3) Start API
export OPENAI_API_KEY=<your_key>
uvicorn app.api:app --reload --port 8000

# 4) Start UI (new terminal, same venv)
export API_BASE="http://127.0.0.1:8000"
python -m streamlit run app/streamlit_app.py --server.port 8503

Optional: set PINECONE_API_KEY + PINECONE_INDEX to enable Pinecone indexing & retrieval.

Features

  • RAG: FAISS local KB + optional Pinecone namespaces.
  • Summarization: clean prompts, bullets, optional CSV table.
  • Extractors: PyMuPDF/pdfplumber for PDFs & tables, Tesseract OCR (images, scanned PDFs), Whisper for video audio.
  • UX niceties: sticky Pinecone namespace, source citations, photo summary tab.
  • Quality: RAGAS eval, synthetic data helpers, tests.
